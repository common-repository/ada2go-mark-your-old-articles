== Description ==

Marks articles when they are older than 365 days (by default). 

Setup your css for the Message-Box and the Text in the Customizer! You just can change the time periode directly in the settings. If you need help, contact me via https://ada2go.de/

== Frequently Asked Questions ==

= How can i change the style? =

You can change the style in the customizer and use the classes .older-post-text and .older-post-hint!

= How can i change the time for say what is "old"? =

You can change the timecode directly in the settings.

= How can I change the displayed message? =

You can change the displayed message directly in the settings.
